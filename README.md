# TypeScript Parser Combinator & Brainfuck Parser

A dependency-free, from-scratch parser combinator library written in TypeScript. This project includes a core set of parsing utility functions and demonstrates their usage by implementing a recursive parser for the Brainfuck programming language.

## Features

* Zero dependencies, built with standard TypeScript.
* Includes core combinators (and, or, many, some).
* Supports recursive parsing for nested grammars (e.g., Brainfuck loops).
* Generates Abstract Syntax Trees (AST).
